### How to clone project

#### 1. Clone repository

```
git clone https://github.com/LaksCastro/anime-dart.git
```

#### 2. Change directory

```
cd anime-dart
```

#### 3. Install dependencies
```
flutter pub get
```

#### 4. Debug Mode
Press F5 if Vscode or run in Command-Line:
```
flutter run
```

#### 5. To build

```
flutter build apk --split-per-abi
```
